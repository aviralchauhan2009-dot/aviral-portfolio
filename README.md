# Aviral Chauhan's Portfolio

Welcome to the portfolio website of Aviral Chauhan! This website showcases various interests and projects, including sports, fictional books, calisthenics, and movies. 

## Features

- **Parallax Scrolling**: Experience a smooth parallax effect as you scroll through the sections.
- **Animations**: Enjoy various motion effects that enhance the visual appeal of the website.
- **Responsive Design**: The website is designed to be fully responsive, ensuring a great experience on both desktop and mobile devices.

## Sections

- **Sports**: Explore Aviral's passion for sports, including articles, images, and animations.
- **Fictional Books**: Discover Aviral's favorite fictional books with engaging content and visuals.
- **Calisthenics**: Learn about calisthenics, including workout routines and tips.
- **Movies**: Dive into Aviral's movie recommendations and reviews.
- **Projects**: Stay tuned, I will add soon.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/aviral-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd aviral-portfolio
   ```
3. Open the `index.html` file in your web browser to view the portfolio.

## Technologies Used

- HTML5

- Parallax.js for parallax scrolling effects
